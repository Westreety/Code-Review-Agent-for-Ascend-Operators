# Code-Review-Agent-for-Ascend-Operators
We are currently designing a Code Review Agent​ project for AscendC​ Huawei Ascend operators. The goal is to detect potential bugs in operators developed by third-party developers, with plans to integrate it into the complete engineering workflow in the future.

我们目前正在完成一个面向华为昇腾（AscendC）算子的代码审查智能体（Code Review Agent）项目。该项目旨在发现第三方开发者所编写算子中的潜在缺陷（Bug），并计划在未来将其集成至完整的工程研发工作流中。
